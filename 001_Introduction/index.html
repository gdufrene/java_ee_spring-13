Consignes pour le projet

Créer un compte Github par personne,
Créer un projet et donner les droits à vos contributeurs (collègues).
L'activité sur github doit être représentative de ce que chacun a fait.
De préférence, utilisez maven. La compilation doit être simple.
Doit pouvoir créer un War et le déployer dans Jetty, Tomcat ou autre ...
Ecrivez un README dévrivant :
- votre projet en quelques lignes ;
- comment compiler, lancer votre projet.

Prévoir une "démo" du projet (15 minutes)
Lister les principales difficultés et comment elles ont été résolues. (5 minutes)

Merci !
